# ISS-Teatru
O institutie teatrala pune la dispozitia spectatorilor un sistem pentru rezervarea locurilor la spectacole. În fiecare zi, institutia are o singura reprezentatie, la care spectatorii pot rezerva locuri începand de dimineata. Pentru un loc în sala, sunt memorate urmatoarele informatii: pozitie (rândul x, loja y etc.), numar si pret. Terminalele puse la dispozitia spectatorilor afiseaza întreaga configuratie a salii, precizând pentru fiecare loc pozitia, numarul, pretul si starea (liber sau rezervat). Folosind un astfel de terminal, spectatorul îsi poate introduce datele personale(isi creaza un cont), poate selecta unul sau mai multe locuri si poate declansa un buton pentru rezervarea lor. După fiecare rezervare, toate terminalele vor afisa situația actualizata referitor la ocuparea sălii.
